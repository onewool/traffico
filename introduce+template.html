Traffico는 공공기관에서 제공하는 교통 오픈데이터를 수집해 쉽고 빠르게 데이터 전처리 과정을 거치고, 데이터 시각화를 통해 사용자들이 보다 편리하게 사고 기록을 볼 수 있게 도와주는 서비스입니다.

웹 데이터 수집

Traffico는 Pandas를 통해 공공기관에서 제공하는 교통 오픈데이터 및 openAPI를 정보를 수신하고 데이터를 로드, 인코딩을 지정합니다.

                  "index" : "사고 건 수",
                  

                    "column" : [

                    "사고ID"
                    "발생 일시"
                    "종료 일시"
                    "분류 코드""
                    "분류 명"
                    "메시지"
                    "동경"
                    "북위"
                    ],
                    "data" :  [
                    "발생 일시"
                    "분류명"
                    "경도"
                    "위도"
                  ]
}

Pandas를 통한 변수 관리

Traffico는 Pandas를 통해 공공기관에서 제공하는 교통 오픈데이터 및 openAPI를 정보를 수신하고 데이터를 로드, 인코딩을 지정합니다.
  • 소스를 수정하지 않고 모든 태그를 관리
  • 맞춤 함수를 사용 및 체이닝 기법을 통해 목적에 맞는 수집항목 태깅
  • requests모듈을 통해 raw.github 데이터 불러오기
  • pivot, transpose와 같은 reshape 기능
  • groupby, pibot_table등을 통한 데이터 요약과 분석

Numpy를 통한 데이터 타입 관리

Traffico는 Numpy를 통해 기본적인 수치계산을 하고, 데이터 형태를 지정 및 변경합니다.
  • 비교적 빠른 연산을 지원하며 메모리를 효율적으로 사용
  • astype함수를 통해 데이터 형태 변환
  • group화를 위해 round함수를 통한 위도 경도 반올림
  • where, select함수를 활용한 조건식 작성

Matplotlib를 통한 데이터 시각화

Traffico는 Matplotlib를 통해 boxplot, scatter, barplot 등 기본적인 시각화 방법을 사용합니다.
  • boxplot을 통한 이상치 검증
  • scatter를 통한 분포도 확인
  • barplot을 통한 사건 수 비교 및 분기별 건수 변화 비교
  • pieplot을 통한 원인에 따른 건수 비교

Folium을 통한 지도 데이터 활용

Traffico는 Folium의 OpenStreetMap 및 공공 GeoJson 데이터를 통해 지도 데이터를 활용합니다.
  • folium모듈을 사용해 지도데이터 불러오기
  • GeoJson으로 대한민국 행정구역 시도, 시군구에 대한 공간 데이터 불러오기
  • 위경도 정보로 marker, circlemarker를 활용 choropleth 그리기
  • 원으로 범위를 표기해 각 시도별 사고건 수 시각화
  • html 파일로 내보내기 기능 수행
